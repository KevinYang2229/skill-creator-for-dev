## 移動平均線（MA）

### 觀察週期

| 均線 | 代表意義         | 短線重要性 |
| ---- | ---------------- | ---------- |
| 5MA  | 週線，極短線趨勢 | ★★★★★      |
| 10MA | 雙週線，短線趨勢 | ★★★★☆      |
| 20MA | 月線，中短線趨勢 | ★★★★☆      |
| 50MA | 季線，中期趨勢   | ★★★☆☆      |

### 先決條件（Phase 2）

- 當日股價必須**高於 20MA（月線）且高於 50MA（季線）**，否則不納入篩選

### 多空判斷

| 訊號     | 條件                         | 解讀           |
| -------- | ---------------------------- | -------------- |
| 多頭排列 | 5MA > 10MA > 20MA > 50MA     | 強勢上漲趨勢   |
| 空頭排列 | 5MA < 10MA < 20MA < 50MA     | 弱勢下跌趨勢   |
| 黃金交叉 | 短期均線由下往上穿越長期均線 | 短線偏多訊號   |
| 死亡交叉 | 短期均線由上往下穿越長期均線 | 短線偏空訊號   |
| 均線糾結 | 多條均線靠近纏繞             | 即將變盤，觀望 |

### 支撐壓力

- 股價站上 5MA → 極短線偏多
- 股價站上 20MA → 短線偏多
- 股價跌破 20MA → 短線轉弱
- 均線斜率向上 → 趨勢偏多；均線走平或下彎 → 動能減弱

## 隨機指標（KDJ）

### 參數設定

- 預設：9 日 KDJ（短線常用）
- K 值為快線，D 值為慢線，**J 值 = 3K − 2D**

### K 值訊號判讀

| K 值區間    | 顯示格式 | 解讀             |
| ----------- | -------- | ---------------- |
| K ≥ 80      | 紅色字體 | 超買區，留意回檔 |
| 20 < K < 80 | 黑色字體 | 正常區間         |
| K ≤ 20      | 綠色字體 | 超賣區，留意反彈 |

### J 值訊號判讀

| J 值區間    | 顯示格式     | 解讀               |
| ----------- | ------------ | ------------------ |
| J ≥ 100     | 紅色「超買」 | 極度超買，短線過熱 |
| 20 < J < 80 | 黑色數值     | 正常區間           |
| J ≤ 20      | 綠色「超賣」 | 極度超賣，留意反彈 |

### 交叉訊號

| 訊號     | 條件                   | 解讀                     |
| -------- | ---------------------- | ------------------------ |
| 黃金交叉 | 昨日 K < D，今日 K > D | 短線買進訊號             |
| 死亡交叉 | 昨日 K > D，今日 K < D | 短線賣出訊號             |
| 無訊號   | 排除以上，以 `–` 表示  | 無明確方向               |
| 高檔鈍化 | KD 持續在 80 以上      | 強勢股特徵，不宜過早放空 |
| 低檔鈍化 | KD 持續在 20 以下      | 弱勢股特徵，不宜過早抄底 |

### Phase 2 顯示格式

```
KDJ 欄位格式：{訊號}\n{K值}, {D值}, {J值}
範例：黃金交叉\n45, 38, 59
範例：超買\n86, 76, 105
範例：–（無訊號時）
```

### 背離訊號

- **頂背離**：股價創新高，但 KD 未創新高 → 上漲動能衰竭
- **底背離**：股價創新低，但 KD 未創新低 → 下跌動能衰竭

## MACD

### 組成

- **DIF（快線）**：12 日 EMA − 26 日 EMA
- **MACD（慢線）**：DIF 的 9 日 EMA
- **柱狀體（OSC）**：DIF − MACD

### 訊號判讀

| 訊號       | 條件                  | 解讀                     |
| ---------- | --------------------- | ------------------------ |
| 多方確認   | DIF > 0 且 MACD > 0   | 中期趨勢偏多             |
| 空方確認   | DIF < 0 且 MACD < 0   | 中期趨勢偏空             |
| 黃金交叉   | DIF 由下往上穿越 MACD | 買進訊號（零軸上方更強） |
| 死亡交叉   | DIF 由上往下穿越 MACD | 賣出訊號（零軸下方更強） |
| 柱狀體翻正 | OSC 由負轉正          | 多方動能轉強             |
| 柱狀體翻負 | OSC 由正轉負          | 多方動能轉弱             |

### Phase 2 簡化顯示模式

在條件篩選（流程二）中，MACD 採簡化顯示：

- 有交叉訊號 → 顯示「黃金交叉」或「死亡交叉」
- 無明確訊號 → 顯示 `–`

### 背離訊號

- **頂背離**：股價創新高，MACD 柱狀體縮減 → 漲勢將盡
- **底背離**：股價創新低，MACD 柱狀體縮減 → 跌勢將盡

## 相對強弱指標（RSI）

### 參數設定（Phase 2 更新）

- 統一使用 **9 日 RSI（9T）**

### 訊號判讀

| RSI 9T 區間  | 顯示格式 | 解讀                         |
| ------------ | -------- | ---------------------------- |
| ≥ 80         | 紅色字體 | 極度超買，短線獲利了結風險高 |
| 20 < 9T < 80 | 黑色字體 | 中性區間                     |
| ≤ 20         | 綠色字體 | 極度超賣，可能出現技術性反彈 |

### 背離

- **頂背離**：股價新高但 RSI 下降 → 賣壓增加
- **底背離**：股價新低但 RSI 上升 → 買盤出現

## 布林通道

### 組成

- **中軌**：20 日移動平均線
- **上軌**：中軌 + 2 倍標準差
- **下軌**：中軌 − 2 倍標準差

### 訊號判讀

| 訊號       | 條件                   | 解讀                   |
| ---------- | ---------------------- | ---------------------- |
| 觸及上軌   | 股價碰到或突破上軌     | 短線過熱，留意回檔風險 |
| 觸及下軌   | 股價碰到或跌破下軌     | 短線超跌，留意反彈機會 |
| 通道收窄   | 上下軌靠近（帶寬縮小） | 盤整即將結束，準備變盤 |
| 通道擴大   | 上下軌擴散             | 趨勢加速，波動加大     |
| 沿上軌行走 | 股價持續貼著上軌       | 強勢上漲，趨勢明確     |
| 沿下軌行走 | 股價持續貼著下軌       | 弱勢下跌，不宜搶反彈   |

### %B 指標

- %B = (收盤價 − 下軌) / (上軌 − 下軌)
- %B > 1 → 突破上軌；%B < 0 → 跌破下軌

## 成交量分析

### 量價關係

| 模式     | 股價 | 成交量 | 解讀                       |
| -------- | ---- | ------ | -------------------------- |
| 價漲量增 | ↑    | ↑      | 正常上漲，趨勢健康         |
| 價漲量縮 | ↑    | ↓      | 上漲乏力，留意回檔         |
| 價跌量增 | ↓    | ↑      | 賣壓湧出，短線偏空         |
| 價跌量縮 | ↓    | ↓      | 賣壓減緩，可能止跌         |
| 爆量長紅 | ↑↑   | ↑↑↑    | 突破訊號（需確認是否有效） |
| 爆量長黑 | ↓↓   | ↑↑↑    | 出貨訊號，短線迴避         |

### 量能指標

- **5 日均量**：近一週平均量能，判斷短線量能水準
- **量比**：當日成交量 / 5 日均量，> 1.5 為放量，< 0.5 為縮量
- **連續放量**：連續 3 日以上量增 → 資金持續關注

### Phase 2 量能判斷

- **篩選條件**：今日成交量 > 5日均量 × 1.5 → 標記 ⭕
- 不符合則不顯示標記

## 5日前高判斷

### 定義

當天收盤價**皆高於前 5 個交易日**的收盤價。

### 顯示規則

- 符合 → 標記 ⭕
- 不符合 → 不顯示

### 意義

- 代表短期內股價呈現明確上升趨勢
- 搭配量能放大更具參考價值

## 乖離率（BIAS）

### 公式

```
乖離率 = (收盤價 − N日MA) / N日MA × 100%
```

### 常用週期

| 週期  | 用途                   |
| ----- | ---------------------- |
| 5 日  | 極短線乖離             |
| 10 日 | 短線乖離               |
| 20 日 | 中短線乖離（主要參考） |

### 判讀原則

| 乖離率區間 | 解讀                     |
| ---------- | ------------------------ |
| > +10%     | 短線嚴重超漲，回檔風險高 |
| +5% ~ +10% | 偏多但接近過熱           |
| -5% ~ +5%  | 正常區間                 |
| -10% ~ -5% | 偏空但可能止跌           |
| < -10%     | 短線嚴重超跌，反彈機會高 |

### Phase 2 顯示格式

- 直接顯示乖離率百分比數值，例如 `20.19%`

## 10日漲幅

### 定義

近 10 個交易日的累計漲跌幅百分比。

### 計算方式

```
10日漲幅 = (今日收盤價 − 10日前收盤價) / 10日前收盤價 × 100%
```

### Phase 2 顯示格式

```
{漲幅百分比}
{天數}天
範例：24.53%\n10天
```

### 判讀原則

- 正值代表近期上漲趨勢
- 高漲幅（>20%）搭配超買指標需留意回檔
- 負值代表近期走弱

## 綜合研判邏輯

### 短線做多條件（至少符合 4 項以上）

1. 股價站上 5MA 且 5MA 向上
2. KDJ 低檔黃金交叉（K > D 且 K < 50）
3. MACD 柱狀體由負轉正或持續擴大
4. RSI 9T 介於 40 ~ 70 區間（上升趨勢中）
5. 成交量放大（今日成交量 > 5日均量 × 1.5）
6. 股價未觸及布林上軌（仍有上漲空間）
7. 收盤價破5日前高
8. 乖離率未超過 +10%

### 短線偏空條件（至少符合 4 項以上）

1. 股價跌破 5MA 且 5MA 下彎
2. KDJ 高檔死亡交叉（K < D 且 K > 50）
3. MACD 柱狀體由正轉負或持續縮減
4. RSI 9T 介於 30 ~ 60 區間（下降趨勢中）
5. 成交量萎縮（量比 < 0.8）
6. 股價接近或跌破布林下軌
7. 乖離率低於 -10%
