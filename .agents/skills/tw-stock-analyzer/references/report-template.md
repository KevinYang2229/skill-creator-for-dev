## 綜合評分體系

### 三構面權重

| 構面   | 短線權重 | 說明                     |
| ------ | -------- | ------------------------ |
| 技術面 | 40%      | 短線操作最重要的判斷依據 |
| 籌碼面 | 35%      | 資金流向直接影響短線走勢 |
| 基本面 | 25%      | 提供安全邊際與中期支撐   |

### 綜合分數計算

```
綜合分數 = 技術面分數 × 0.4 + 籌碼面分數 × 0.35 + 基本面分數 × 0.25
```

### 評等對照

| 分數區間 | 評等          | 操作建議             |
| -------- | ------------- | -------------------- |
| 80 ~ 100 | A（強力推薦） | 積極做多，短線看漲   |
| 65 ~ 79  | B（偏多操作） | 可考慮進場，設好停損 |
| 50 ~ 64  | C（中性觀望） | 等待更明確訊號       |
| 35 ~ 49  | D（偏空迴避） | 不宜進場，觀望為主   |
| 0 ~ 34   | E（強力迴避） | 短線風險高，迴避     |

## Phase 2 條件篩選規則

### 先決條件

| 條件       | 規則                                                                 |
| ---------- | -------------------------------------------------------------------- |
| 股價範圍   | 30 ~ 250 元                                                          |
| 均線位置   | 當日股價高於 20MA（月線）且高於 50MA（季線）                         |
| 殖利率     | > 1%                                                                 |
| EPS        | 比前一季高                                                           |
| 排除交易量 | ≤ 1000 張                                                            |
| 排除類股   | 建材營造、生技、玻璃、造紙、橡膠、運動休閒、居家生活、綠能環保、證券 |

### 比較表欄位格式計算規則

- **破5日前高**：若當日收盤價高於前 5 天高點，標記 `⭕`；否則不顯示。
- **乖離率**：填寫近 20日乖離率百分比。
- **KDJ / MACD**：填寫指標特徵，如 `黃金交叉`、`死亡交叉`、`超買`、`超賣` 等；無明顯訊號以 `–` 表示。

## 操作建議設定原則

### 停損停利設定原則

| 策略           | 停損幅度  | 第一停利  | 第二停利     |
| -------------- | --------- | --------- | ------------ |
| 積極型（A 級） | -3% ~ -5% | +5% ~ +8% | +10% ~ +15%  |
| 穩健型（B 級） | -3% ~ -5% | +3% ~ +5% | +8% ~ +10%   |
| 保守型（C 級） | -2% ~ -3% | +3% ~ +5% | 不設第二停利 |

### 持有天數建議

- 極短線（當沖~隔日沖）：1 ~ 2 日
- 短線波段：3 ~ 10 個交易日
- 短中期波段：10 ~ 20 個交易日
