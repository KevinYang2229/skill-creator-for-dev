## 營收分析

### 核心指標

| 指標           | 計算方式                                    | 說明                   |
| -------------- | ------------------------------------------- | ---------------------- |
| 月營收         | 公司每月公布                                | 10 日前公布上月營收    |
| 月增率（MoM）  | (本月 − 上月) / 上月 × 100%                 | 短期動能變化           |
| 年增率（YoY）  | (本月 − 去年同月) / 去年同月 × 100%         | 排除季節性因素的成長率 |
| 累計營收年增率 | (今年累計 − 去年同期累計) / 去年同期 × 100% | 整體成長趨勢           |

### 判讀原則

| 訊號                   | 條件                 | 解讀                   |
| ---------------------- | -------------------- | ---------------------- |
| 營收創歷史新高         | 單月營收超越歷史紀錄 | 強烈利多，短線催化     |
| 連續 3 月以上 YoY 成長 | 營收年增率連續正成長 | 成長趨勢確立           |
| 營收年增率由負轉正     | YoY 從負數轉為正數   | 谷底回升訊號           |
| 營收月增率 > 20%       | 大幅月增             | 短期訂單爆發或旺季效應 |
| 營收年增率連續衰退     | YoY 連續負成長       | 營運動能衰退           |
| 營收低於市場預期       | 低於法人預估值       | 短線利空               |

### 注意事項

- 注意產業**淡旺季效應**，MoM 需搭配歷史同期比較
- 合併營收 vs. 母公司營收的差異
- 一次性大單或認列可能造成營收跳升，需確認延續性

## 獲利能力

### 核心指標

| 指標                 | 計算方式                            | 說明         |
| -------------------- | ----------------------------------- | ------------ |
| 毛利率               | (營收 − 營業成本) / 營收 × 100%     | 產品競爭力   |
| 營益率（營業利益率） | 營業利益 / 營收 × 100%              | 本業獲利能力 |
| 稅後淨利率           | 稅後淨利 / 營收 × 100%              | 整體獲利能力 |
| EPS（每股盈餘）      | 稅後淨利 / 流通在外股數             | 每股獲利     |
| 季 EPS 年增率        | (本季 − 去年同季) / 去年同季 × 100% | 獲利成長性   |

### 判讀原則

| 訊號                | 條件                      | 解讀                             |
| ------------------- | ------------------------- | -------------------------------- |
| 毛利率連續上升      | 近 3 季毛利率上升         | 產品組合改善或漲價成功           |
| 毛利率下滑          | 近 3 季毛利率下降         | 成本上升或競爭加劇               |
| EPS 超越預期        | 高於法人預估 EPS          | 短線利多，上修空間               |
| EPS 低於預期        | 低於法人預估 EPS          | 短線利空，下修風險               |
| EPS 季增長          | 本季 EPS > 前一季 EPS     | 獲利動能提升（Phase 2 先決條件） |
| 營益率 > 毛利率變化 | 營益率改善幅度大於毛利率  | 費用控管得當                     |
| 業外收益佔比高      | 業外損益 / 稅後淨利 > 30% | 獲利品質不穩定                   |

### 觀察週期

- **季報**：每季結束後 45 日內公布（3/31、5/15、8/14、11/14 前）
- **月營收**：每月 10 日前公布
- 短線操作重點關注**最近一季**與**當月營收**趨勢

## 估值指標

### 核心指標

| 指標              | 計算方式          | 說明                   |
| ----------------- | ----------------- | ---------------------- |
| 本益比（PE）      | 股價 / 近四季 EPS | 市場願意付出的獲利倍數 |
| 本淨比（PB）      | 股價 / 每股淨值   | 相對帳面價值的溢價     |
| 本益成長比（PEG） | PE / EPS 年增率   | 考量成長性的合理估值   |

### 判讀原則

| 訊號            | 條件                     | 解讀                     |
| --------------- | ------------------------ | ------------------------ |
| PE 低於歷史平均 | 當前 PE < 近 3 年平均 PE | 相對便宜，可能低估       |
| PE 高於歷史平均 | 當前 PE > 近 3 年平均 PE | 相對昂貴，留意修正       |
| PB < 1          | 股價低於淨值             | 可能低估（需排除虧損股） |
| PEG < 1         | 本益比低於成長率         | 成長股合理估值           |
| PEG > 2         | 本益比遠高於成長率       | 估值偏高                 |

### 產業別 PE 參考

- 不同產業的合理 PE 範圍差異大
- 需與**同產業**公司比較，而非跨產業
- 高成長產業（如 AI、半導體）PE 通常較高

## 產業趨勢與題材

### 分析面向

| 面向     | 內容                        | 短線影響         |
| -------- | --------------------------- | ---------------- |
| 產業循環 | 上升期/高原期/衰退期/谷底期 | 順循環操作勝率高 |
| 政策利多 | 政府政策、補貼、法規鬆綁    | 短線催化劑       |
| 國際趨勢 | 全球供應鏈、原物料價格      | 影響基本面預期   |
| 法人報告 | 研究機構評等調升/調降       | 短線股價催化     |
| 題材面   | AI、電動車、生技等熱門題材  | 短線資金吸引力   |

### 題材面判讀

- **新興題材**：剛出現的市場熱點，資金快速湧入 → 短線爆發力強
- **延續性題材**：持續獲得市場關注的趨勢 → 波段操作機會
- **退潮題材**：市場熱度降低的舊題材 → 避免追高
- 題材需搭配**實質營收貢獻**判斷持續性

## 綜合研判邏輯

### 短線做多條件（至少符合 3 項以上）

1. 近月營收年增率 > 10%
2. 最近一季 EPS 年增率 > 0%
3. 毛利率或營益率趨勢向上
4. 估值（PE）低於或接近歷史平均
5. 具備當前市場熱門題材
6. 法人上修目標價或評等

### 短線偏空條件（至少符合 3 項以上）

1. 營收連續 3 月以上年增率衰退
2. 毛利率與營益率同步下滑
3. EPS 低於市場預期
4. PE 遠高於同產業平均（> 1.5 倍）
5. 題材退潮、市場關注度下降
6. 法人下修目標價或評等

### 基本面評分標準

| 項目       | 多方滿分   | 判斷依據                                               |
| ---------- | ---------- | ------------------------------------------------------ |
| 營收動能   | 30 分      | 年增 > 20% 得 30 / 10~20% 得 20 / 0~10% 得 10 / 衰退 0 |
| 獲利能力   | 25 分      | EPS 成長+毛利率升 25 / 持平 13 / 衰退 0                |
| 估值合理性 | 20 分      | PE < 歷史均值 20 / 接近 10 / 遠高於 0                  |
| 產業題材   | 25 分      | 熱門+有營收 25 / 熱門無營收 13 / 退潮 0                |
| **合計**   | **100 分** |                                                        |
