## 三大法人買賣超

### 法人特性

| 法人   | 操作風格                   | 對短線影響                |
| ------ | -------------------------- | ------------------------- |
| 外資   | 中長線布局，資金規模最大   | ★★★★☆（連續性買賣影響大） |
| 投信   | 季度績效導向，偏好中小型股 | ★★★★★（短線催化效果最強） |
| 自營商 | 短線進出，避險操作多       | ★★★☆☆（波動大但持續性低） |

### 判讀原則

| 訊號            | 條件                       | 解讀                         |
| --------------- | -------------------------- | ---------------------------- |
| 三大法人同買    | 外資 + 投信 + 自營商皆買超 | 強烈偏多訊號                 |
| 外資連買        | 外資連續 3 日以上買超      | 中長線看好，短線有撐         |
| 投信連買        | 投信連續 3 日以上買超      | 投信作帳或看好標的，短線催化 |
| 外資 + 投信同買 | 外資與投信同步買超         | 最佳短線做多訊號             |
| 三大法人同賣    | 外資 + 投信 + 自營商皆賣超 | 強烈偏空訊號                 |
| 外資大量賣超    | 單日賣超佔成交量 > 20%     | 重大利空或獲利了結           |

### 量化指標

- **法人買賣超佔成交量比**：買賣超張數 / 當日成交量 × 100%
  - \> 10%：法人積極進場/出場
  - 5% ~ 10%：法人適度參與
  - < 5%：法人態度中性

## 外資 / 投信 / 自營商持股變化

### 持股比例觀察

| 指標           | 說明                 | 短線意義                    |
| -------------- | -------------------- | --------------------------- |
| 外資持股比例   | 外資持有佔總股本比例 | 比例穩定上升 → 外資看好     |
| 投信持股比例   | 投信基金持有比例     | 比例快速增加 → 投信作帳標的 |
| 外資持股週變化 | 較上週持股增減       | 連續增持 → 偏多佈局         |

### 重點觀察

- **外資持股 > 30%** 且持續增加：大型權值股常見，股價相對穩定
- **投信持股突然大增**：可能是新納入基金或作帳行情
- **外資與投信同步增持**：最強短線支撐訊號
- **自營商持股變化大**：多為避險操作，參考價值較低

## 融資融券

### 指標說明

| 指標       | 計算方式                   | 說明             |
| ---------- | -------------------------- | ---------------- |
| 融資餘額   | 累計融資買進未償還股數     | 代表散戶看多程度 |
| 融券餘額   | 累計融券賣出未回補股數     | 代表市場看空程度 |
| 融資使用率 | 融資餘額 / 融資限額 × 100% | 散戶槓桿程度     |
| 券資比     | 融券餘額 / 融資餘額 × 100% | 軋空潛力指標     |
| 資券互抵   | 當沖交易量                 | 短線投機程度     |

### 判讀原則

| 訊號              | 條件                   | 解讀                   |
| ----------------- | ---------------------- | ---------------------- |
| 融資大增 + 股價漲 | 融資餘額連增且股價上漲 | 散戶追高，留意高檔風險 |
| 融資大增 + 股價跌 | 融資餘額增但股價下跌   | 散戶接刀，底部未明     |
| 融資大減 + 股價跌 | 融資餘額減且股價下跌   | 融資斷頭，恐慌性賣壓   |
| 融資大減 + 股價穩 | 融資餘額減但股價持穩   | 籌碼清洗完成，正面訊號 |
| 券資比 > 30%      | 融券/融資比例高        | 軋空行情機會大         |
| 融券大增          | 融券餘額連續增加       | 市場看空，但有軋空風險 |

## 大戶與散戶持股比例

### 資料來源

- 集保戶股權分散表（每週公布）
- 依持股級距分類（1~999 股、1,000~5,000 股…400 張以上）

### 判讀原則

| 訊號                | 條件                        | 解讀               |
| ------------------- | --------------------------- | ------------------ |
| 大戶增持            | 400 張以上持股比例增加      | 大戶看好，偏多     |
| 散戶增持            | 小額持股（< 10 張）比例增加 | 散戶進場，留意高點 |
| 大戶減持 + 散戶增持 | 籌碼由大戶流向散戶          | 大戶出貨，短線危險 |
| 散戶減少 + 大戶增加 | 籌碼由散戶流向大戶          | 籌碼集中，偏多訊號 |
| 千張大戶持續增加    | 1,000 張以上大戶連續增持    | 主力積極佈局       |

### 籌碼集中度

- **集中度上升**（大戶比例增、散戶比例減）→ 正面
- **集中度下降**（大戶比例減、散戶比例增）→ 負面
- 觀察最近 4 ~ 8 週變化趨勢

## 主力券商分點進出

### 觀察方式

- 查詢個股前 15 大買賣券商分點
- 抓取特定券商分點的連續買賣超紀錄

### 判讀原則

| 訊號             | 條件                          | 解讀               |
| ---------------- | ----------------------------- | ------------------ |
| 特定分點連買     | 同一券商分點連續 3 日以上買超 | 主力積極佈局       |
| 多家分點同買     | 多個分點同步大量買進          | 法人或大戶聯合進場 |
| 特定分點突然大賣 | 先前買超的分點開始大量賣出    | 主力出貨訊號       |
| 前 15 大買超集中 | 少數幾家分點佔大部分買超      | 籌碼集中，主力控盤 |

### 注意事項

- 分點資料需搭配成交量與股價趨勢綜合判斷
- 單日分點資料參考性有限，至少觀察 3 ~ 5 日趨勢
- 隔日沖券商進出視為雜訊，需排除

## 綜合研判邏輯

### 短線做多條件（至少符合 4 項以上）

1. 外資 + 投信近 5 日同步買超
2. 融資未大幅增加（散戶未過度追高）
3. 券資比 > 20%（有軋空潛力）
4. 大戶持股比例連續增加
5. 主力券商分點連續買超
6. 法人買超佔成交量 > 5%

### 短線偏空條件（至少符合 4 項以上）

1. 三大法人連續賣超
2. 融資餘額大幅增加（散戶追高）
3. 大戶持股比例下降、散戶比例上升
4. 主力券商分點轉賣
5. 外資大量賣超（佔成交量 > 15%）
6. 融資使用率偏高
